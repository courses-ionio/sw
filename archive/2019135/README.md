# Παραδοτέο 1

## Εισαγωγή
Ένας απο τους στόχους που έχω σχετικά με το μάθημα της «Τεχνολογίας Λογισμικού» είναι η καλύτερη κατανόηση της έννοιας του λογισμικού και να δοκιμάσω διαφορετικά λογισμικά (ArchLinux, Ubuntu κτλ). Γενικότερα, η ιδέα του open source λογισμικού μου αρέσει πολύ και προσπαθώ όσο περισσότερο μπορώ να το χρησιμοποιώ. Επίσης, η συνεργασία με άλλους συμφοιτητές μου, για την ανάπτυξη λογισμικού, πιστεύω ότι θα μου προσφέρει διάφορες γνώσεις, τόσο σε κοινωνικό επίπεδο όσο και σε προγραμματιστικό, διότι δεν γνωρίζω όλες τις γλώσσες προγραμματισμού. Πιστεύω ότι το μάθημα θα μου δώσει τις απαραίτητες βάσεις πάνω στη σχεδίαση και ανάπτυξη λογισμικού και επιπλέον εμπειρία, που θα αποκτήσω απο τη χρήση διαφορετικών λογισμικών ώστε να έχω μια σφαιρική εικόνα σ'αυτά. Επίσης, πιστεύω ότι τα βίντεο τα οποία θα παρακολουθήσω, θα μου προσφέρουν χρήσιμες πληροφορίες και γνώσεις πάνω στο αντικείμενο του λογισμικού. 

**Ενημέρωση εισαγωγής**

Τα τελευταία χρόνια οφείλω να ομολογήσω ότι χρησιμοποιούσα σχεδόν αποκλειστικά έτοιμο λογισμικό , με εξαίρεση την ενασχόληση μου για ένα μικρό χρονικό διάστημα με τα ArchLinux. Απο το έτοιμο λογισμικό θεωρώ ότι αυτά που έμαθα και ο τρόπος με τον οποίο έκανα το οτιδήποτε σ'αυτά τα λογισμικά, ήταν απερίγραπτα περιοριστικά και προωθούσαν το πως ήθελαν οι εταιρίες να χρησιμοποιούν οι χρήστες το λογισμικό τους. Αυτό που ευελπιστώ μέσω του μαθήματος είναι να μάθω περισσότερα σχετικά με το καλής ποιότητας και ανοιχτού κώδικα λογισμικό. Επίσης, να μάθω πως να χρησιμοποιώ και να ρυθμίζω στις ανάγκες μου λογισμικά στην καθημερινότητα μου και να καταλάβω καλύτερα τις λειτουργίες και τις διαδικασίες που τρέχουν στο παρασκήνιο όσο χρησιμοποιείται το λογισμικό.

# Παραδοτέο 11

## Άσκηση γραμμής εντολών - Twilio

Για το 11ο παραδοτέο, υλοποίησα μια απο τις ασκήσεις απο τη κατηγορία του software και συγκεκριμένα το εργαλείο [twilio-python](https://github.com/twilio/twilio-python). Μου φάνηκε ενδιαφέρον διότι χρησιμοποιεί python scripts για να πραγματοποιήσει κλήση και αποστολή μηνύματος στο κινητό τηλέφωνο.

Αρχικά, έπρεπε να δημιουργήσω λογαριασμό στο [Twilio](https://www.twilio.com/) και να συνδέσω σ'αυτόν τον αριθμό του τηλεφώνου μου. Στη συνέχεια, μου δόθηκε ένας αριθμός Twilio ο οποίος θα ήταν ο αριθμός που θα έκανε την κλήση και θα έστελνε μήνυμα στον αριθμό που δήλωσα στην αρχή. Το account_sid και auth_token μου ήταν χρήσιμα για την δημιουργία και των δύο python scripts, ένα για κλήση και ένα για μήνυμα. Η εγκατάσταση του Twilio έγινε με την εντολή `pip install twilio`, σε Alpine Linux.

## Επίλογος

Τελειώνοντας την εργασία του εξαμήνου, πιστεύω ότι απέκτησα πολλές και ενδιαφέρουσες γνώσεις πάνω στο αντικείμενο του λογισμικού. Αρχικά, τα βίντεο κουίζ μου έδωσαν μια διαφορετική οπτική γωνία για τον κόσμο των λογισμικών. Ο απεριόριστος αριθμός Linux distributions που υπάρχουν χρησιμοποιείται απο ένα πολύ μικρό πλήθος χρηστών σε συγκρισή βέβαια με τα πιο διάσημα και εμπορικά λογισμικά, όπως είναι τα Windows και τα MacOS. Γεγονός αξιοσημειώτο, αφού λογισμικά όπως το plan9 που είναι σαφώς καλυτέρο και πιο μοντέρνο, δεν είχαν την ίδια τύχη με τα Windows και τα MacOS λόγω της έλλειψης μάρκετινγκ και της συνήθειας των χρηστών στα ήδη υπάρχοντα λειτουργικά συστήματα. Επίσης, μέσω της εγκατάστασης νέου λειτουργικού συστήματος χωρίς systemd, Alpine Linux, κάταλαβα καλύτερα πως λειτουργεί και τελικά πως να το χρησιμοποιώ σωστά για τις δικές μου ανάγκες. Τέλος, μέσω της όμαδα, κατάλαβα πόσο σημαντικό είναι το κάθε μέλος να έχει έναν ρόλο σ'αυτήν και να κατανοεί τους κανόνες που έχουν τεθεί απο την ομάδα. Με την ομάδα, καταφέραμε να συνεργαστούμε σε αρκετά καλό επίπεδο για την δημιουργία του οργανισμού μας και για οποιοδήποτε εμπόδιο ή προβληματισμό υπήρχε, το συζητούσαμε είτε μέσω της πλατφόρμας του Github είτε με άλλα μέσα επικοινωνίας. Συνοψίζοντας, το μάθημα με σύστησε σε καινούργια εργαλεία, κατάλαβα το πόσο σημαντικό είναι να μην αποκλείω διαφορέτικα εργαλεία και λειτουργικά συστήματα επειδή έχω συνηθήσει να χρησιμοποιώ συγκεκριμένα εργαλειά και λειτουργικά συστήματα και βέβαια πόσο καθοριστικό ρόλο έχει μια ομάδα στη δημιουργία λογισμικού.

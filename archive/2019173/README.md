# Εισαγωγή (1ο Παραδοτέο)
Στην Πληροφορική όσες περισσότερες γνώσεις έχει κάποιος τόσο πιο καλός θα είναι στην δουλειά του. Για τον λόγο αυτό, θα ήθελα μέσα από αυτό το μάθημα να αποκτήσω εμπειρία, δουλεύοντας με λογισμικά, χωρίς systemd, και φυσικά να γνωρίσω καλύτερα πως λειτουργεί ο κόσμος των Πληροφορικών, όταν συνεργάζονται για να βγει ένα συλλογικό αποτέλεσμα. Συγκεκριμένα, θα ήθελα να μην χρησιμοποιώ πλέον τις βοηθητικές ρόδες όπως πολύ ωραία έθεσε ο Άλαν Κέη και θα είμαι ικανή να χρησιμοποιώ συστήματα χωρίς systemd, όπως το freebsd, το kiss, το void linux και το funtoo. Επίσης, ξέρουμε πως όσες γνώσεις και δεξιότητες και αν έχει κάποιος, δεν μπορεί να λύσει όλα τα προβλήματα, που θα κληθεί να αντιμετωπίσει. Για αυτό, οι δεξιότητες συνεργασία και ομαδικότητα, όπου καλούμαστε να χρησιμοποιήσουμε στην εργασία μας, αποτελούν κύρια συστατικά της επιτυχίας, μίας επιτυχίας που το βάρος της πέφτει και στις δύο μεριές(αρχηγός-μέλη ομάδας). Έτσι, δίνεται η ευκαιρία με το μάθημα Τεχνολογία Λογισμικού, τους στόχους, που έχουμε να εκπληρώσουμε μέσα από την εργασία, που καλούμαστε να κάνουμε και τα εβδομαδιαία βίντεο κουίζ, να εμπλουτίσω τις γνώσεις μου πάνω σε εργαλεία λογισμικού, που θα μου χρειαστούν σε κάποια μελλοντική δουλειά, και να αναπτύξω τις δεξιότητές μου μέσα από την συνεργασία και την ομαδικότητα για την έκβαση ενός ολοκληρωμένου αποτελέσματος.

# Άσκηση γραμμής εντολών (4ο Παραδοτέο)

Στο συγκεκριμένο παραδοτέο μας ζητήθηκε να κάνουμε μία άσκηση γραμμής εντολών με ένα λογισικό, το οποίο θα είναι χωρίς systemd. Έτσι, εγκατέστησα το λογισμικό Void Linux και επέλεξα την άσκηση organise the terminal window into multiple areas από την κατηγορία [warmup](https://github.com/epidrome/dokey#warmup). Συγκεκριμένα, η άσκηση αυτή ζητάει να χρησιμοποιηθεί ένα παράθυρο για αναζήτηση/επεξεργασία τοπικών αρχείων ή περιήγηση στον Ιστό και ένα άλλο παράθυρο για παρακολούθηση απόδοσης. Όπως θα δείτε και από το asciinema μου έκανα δύο αναζητήσεις στον ιστό και παρακολούθησα την απόδοση και τις διεργασίες πίσω από αυτές τις αναζητήσεις. Ήθελα να πραγματοποιήσω αυτήν την άσκηση, καθώς με την ολοκλήρωσή της θα μάθαινα τι διαδικασίες γίνονται πίσω από μία αναζήτηση στο διαδίκτυο. Για την ολοκλήρωση αυτής της άσκησης χρησιμοποίησα το τερματικό του Void Linux. Αναλυτικότερα, έπρεπε:
- να κατεβάσω το glances, χρησιμοποιώντας την εντολή sudo xbps-install -S glances,
- να πατήσω E, ώστε να γίνει reset και
- z, ώστε να απεικονιστούν οι διεργασίες.

Παρακάτω είναι χρήσιμοι σύνδεσμοι:

[Εδώ](https://github.com/nicolargo/glances) είναι το link της ιστοσελίδας που με βοήθησε.

Θεωρώ πως κατάφερα να εκπληρώσω τις επιθυμίες μου ως προς τον εμπλουτισμό της γνώσης μου πάνω στο συγκεκριμένο αντικειμένο και πως είμαι ικανή να χρησιμοποιώ και να επεξεργάζομαι υψηλής απαίτησης λογισμικά.
